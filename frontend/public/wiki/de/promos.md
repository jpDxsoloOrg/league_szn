# Promos

*Diese Funktion ist für Wrestler und kann von Ihrer Liga ein- oder ausgeschaltet werden.*

Schneiden Sie Promos, um Storylines aufzubauen, Gegner herauszufordern oder anderen Wrestlern zu antworten. Promos erscheinen im Feed und können mit Herausforderungen verknüpft werden.

## Promo-Feed

Öffnen Sie **Promos** im Menü für den Feed. Angepinnte Promos stehen oben. Scrollen Sie für Open-Mic-Promos und Call-Outs. Um Antworten auf eine Promo zu lesen, öffnen Sie die Promo — die Thread-Ansicht zeigt das Original und alle Antworten.

## Eine Promo erstellen

Klicken Sie auf **Promo schneiden**, um den Editor zu öffnen. Geben Sie Ihrer Promo einen Titel und schreiben Sie den Text. Nutzen Sie **@WrestlerName**, um andere Wrestler zu erwähnen.

Wählen Sie einen Typ: **Open Mic** (allgemeine Promo), **Call-Out** (Wrestler herausfordern; kann eine Herausforderung erzeugen) oder **Response** (auf die Promo eines anderen Wrestlers antworten).

## Auf Promos reagieren

Bei jeder Promo können Sie eine Reaktion hinterlassen. Reaktionen sind für alle sichtbar und bereichern die Storyline.

## Call-Outs und Herausforderungen

Eine Call-Out-Promo kann automatisch eine Herausforderung erzeugen. Der herausgeforderte Wrestler kann annehmen, ablehnen oder kontern. Sobald ein Match aus einer Herausforderung geplant ist, erscheint es ggf. auf der Event-Card und in Ihrer Kampf-Historie.
