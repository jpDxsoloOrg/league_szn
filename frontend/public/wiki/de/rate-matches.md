# Kämpfe bewerten

Sobald ein Kampf von einem GM als abgeschlossen markiert wurde, können angemeldete Nutzer ihn mit 0,5 bis 5 Sternen bewerten.

## So funktioniert es

- Öffne einen abgeschlossenen Kampf (Veranstaltungen → eine Veranstaltung wählen oder die Kampf-Suche verwenden).
- Bewege den Mauszeiger über die Sternleiste unter dem Ergebnis, um eine Bewertung vorzusehen, und klicke, um sie zu bestätigen.
- Jeder Nutzer kann einen Kampf **einmal** bewerten. Nach der Abgabe wird das Widget gesperrt und zeigt deine Bewertung.
- Die Gesamtbewertung eines Kampfes ist der Durchschnitt aller abgegebenen Nutzerbewertungen, gerundet auf den nächsten halben Stern.

## Warum es zählt

Wenn ein Kampf zu einer Rivalität gehört, beeinflusst jede deiner Bewertungen die **Hitze** der Rivalität. Hoch bewertete Kämpfe heizen eine Rivalität auf; schlecht bewertete kühlen sie ab.
