# Herausforderungen

*Diese Funktion ist für Wrestler. Sie kann von Ihrer Liga ein- oder ausgeschaltet werden.*

Als Wrestler können Sie Herausforderungen aussprechen, annehmen, ablehnen oder kontern, um Matches mit anderen Spielern zu vereinbaren.

## Herausforderungs-Board

Öffnen Sie **Herausforderungen** im Menü, um das Board zu sehen. Es listet aktive Herausforderungen, die Sie gesendet oder erhalten haben. Nutzen Sie die Filter (Aktiv, Ausstehend, Angenommen, Zuletzt).

## Eine Herausforderung aussprechen

Klicken Sie auf **Herausforderung aussprechen**, um den Promo-Editor im Call-Out-Modus zu öffnen. Schreiben Sie eine kurze Call-Out-Promo und senden Sie sie ab; die Herausforderung wird erstellt und mit Ihrer Promo verknüpft. Sie können auch unter Promos eine Call-Out-Promo erstellen.

## Auf eine Herausforderung reagieren

Öffnen Sie eine Herausforderung im Board oder unter Mein Profil. Sie können **Annehmen** (Match planen), **Ablehnen** oder **Kontern** (Gegenherausforderung mit eigener Stipulation).

## Status von Herausforderungen

Herausforderungen können **Ausstehend**, **Angenommen**, **Abgelehnt**, **Konter**, **Geplant**, **Abgelaufen** oder **Storniert** sein.
