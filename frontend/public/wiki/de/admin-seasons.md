# Saisons verwalten

Die Seite **Saisons** ermöglicht es, die Liga in getrennte Wettbewerbszeiträume zu gliedern.

## Saison anlegen

1. Gehe zu **Saisons**
2. Fülle die Angaben aus:
   - **Saisonname** — z. B. „Saison 1“, „Sommer 2025“
   - **Startdatum** — Wann die Saison beginnt
   - **Enddatum** (optional) — Wann die Saison endet
3. Klicke auf **Neue Saison anlegen**, um zu speichern

## Wichtig: Nur eine aktive Saison

Es kann nur eine Saison gleichzeitig aktiv sein. Du musst die aktuelle Saison beenden, bevor du eine neue anlegst.

## Saison beenden

1. Finde die aktive Saison (mit dem Banner „Aktive Saison“)
2. Klicke auf **Saison beenden**
3. Bestätige die Aktion im Dialog
4. Der Status der Saison wechselt auf „Abgeschlossen“ und das Enddatum wird gespeichert

## Saison löschen

1. Finde die Saison in der Übersicht
2. Klicke auf **Löschen** bei der Saison
3. Bestätige die Aktion im Dialog

**Warnung:** Das Löschen einer Saison löscht dauerhaft alle Tabellenstände dieser Saison. Diese Aktion kann nicht rückgängig gemacht werden.

## So funktionieren Saisons

- Matches können beim Planen einer Saison zugeordnet werden
- Saison-Tabellen werden getrennt von den Gesamttabellen geführt
- Beim Eintragen eines Match-Ergebnisses werden Gesamt- und Saison-Tabellen automatisch aktualisiert
- Abgeschlossene Saisons bleiben als Historie einsehbar
