# Meisterschaften verwalten

Die Seite **Meisterschaften** ermöglicht das Anlegen und Verwalten von Titeln.

## Meisterschaft anlegen

1. Gehe zu **Meisterschaften**
2. Fülle die Angaben aus:
   - **Meisterschaftsname** — z. B. „World Heavyweight Championship“
   - **Typ** — Einzel oder Tag Team
   - **Gürtelbild** (optional) — Lade ein Bild des Meisterschaftsgürtels hoch
3. Klicke auf **Meisterschaft anlegen**, um zu speichern

## Meisterschaft bearbeiten

1. Finde die Meisterschaft in der Tabelle
2. Klicke auf **Bearbeiten**
3. Passe Name, Typ oder Bild an
4. Klicke auf **Speichern**, um die Änderungen zu übernehmen

## Meisterschaft löschen

1. Finde die Meisterschaft in der Übersicht
2. Klicke auf **Löschen** bei der Meisterschaft
3. Bestätige die Aktion im Dialog

**Warnung:** Das Löschen einer Meisterschaft löscht dauerhaft die gesamte Meisterschafts-Historie (vergangene Regentschaften, Daten usw.).

## So funktionieren Meisterschaften

- Beim Planen eines Titelmatches wählst du den auf dem Spiel stehenden Titel
- Wird das Ergebnis eingetragen und der Herausforderer gewinnt, wird er neuer Meister
- Die Meisterschafts-Historie wird automatisch mit Daten und Regentschaftslänge geführt
- Herausforderer-Ranglisten können so konfiguriert werden, dass #1-Herausforderer pro Meisterschaft automatisch ermittelt werden
