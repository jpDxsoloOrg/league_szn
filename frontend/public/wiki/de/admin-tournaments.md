# Turniere anlegen

Die Seite **Turniere** ermöglicht das Anlegen und Verwalten von Turnieren.

## Turnier anlegen

1. Gehe zu **Turniere**
2. Fülle die Angaben aus:
   - **Turniername** — z. B. „King of the Ring 2025“
   - **Typ** — K.-o.-System oder Rundensystem
   - **Teilnehmer** — Wähle die teilnehmenden Spieler
3. Klicke auf **Turnier anlegen**, um zu speichern

## Turnier-Typen

### K.-o.-System (Single Elimination)

- Mindestens 4 Teilnehmer
- Das Bracket wird automatisch erstellt
- Verlierer scheiden aus
- Sieger rücken in die nächste Runde vor
- Gut für kurze Turniere

### Rundensystem (Round Robin)

- Mindestens 2 Teilnehmer
- Jeder spielt gegen jeden
- Punkte: Sieg = 2, Unentschieden = 1, Niederlage = 0
- Die meisten Punkte gewinnen
- Gut für ligaähnliche Formate

## Turnier-Matches verwalten

1. Nach dem Anlegen des Turniers plane Matches über **Match planen**
2. Wähle beim Planen das Turnier im Dropdown aus
3. Trage die Ergebnisse ein, sobald die Matches stattgefunden haben
4. Standings/Brackets werden automatisch aktualisiert
