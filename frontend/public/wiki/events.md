# Events & PPV

View upcoming and past events, including PPVs, weekly shows, and special events.

## Browsing Events

Events are organized by type. Use the filter tabs to find specific types:

- **PPV** — Major events like WrestleMania, SummerSlam, Royal Rumble
- **Weekly** — Regular weekly shows like Raw, SmackDown, NXT
- **Special** — One-off special events
- **House Show** — Non-televised exhibition events

## Event Details

Click on any event to see its full details:

- The full match card with all scheduled bouts
- Championship matches highlighted with a special badge
- Event date, venue, and description
- Results for completed events with match outcomes
